# GB/T 7714 顺序编码制 Notes

Use these concise rules when normalizing references for this skill.

## In-text citations

- Use bracketed numeric citations such as `[1]`, `[2]`, `[3]`.
- Number by first appearance in正文.
- Reuse the same number every time the same source is cited again.
- Combine multiple sources for one claim as one superscript marker such as `[1,2]` or `[6,9]`. Generated output uses comma style rather than separate `[1][2]` markers.
- Keep citation markers at the end of the supported clause or sentence, before or after punctuation according to the paper's existing style; be consistent within the document.

## Reference list order

- 文末参考文献 must follow正文 first-citation order.
- Do not keep unused references in the final numbered list unless the user explicitly asks.
- Do not invent missing author, title, journal, year, volume, issue, pages, DOI, or URL fields.

## Common formats

Journal:

`[1] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.`

Book:

`[2] 作者. 书名[M]. 出版地: 出版者, 出版年.`

Thesis:

`[3] 作者. 题名[D]. 保存地: 保存单位, 年.`

Conference:

`[4] 作者. 题名[C]//会议论文集名. 出版地: 出版者, 年: 起止页码.`

Online source:

`[5] 作者. 题名[EB/OL]. (发表或更新日期)[引用日期]. URL.`

If an entry is already formatted, preserve its wording and only rewrite the leading number unless the user asks for full formatting cleanup.
